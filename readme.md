# it-sprint1-basics-js

**Descripción**: 
Ejercicios para aprender fundamentos de modern JS (ES6+) con Jest tests.

## Tecnologias

- JavaScript (ES6+)
- Jest
- Node.js

## Instal·lación

1. Clonar repositorio: `git clone https://github.com/bertagonzalezgu/Task-S1.07.git`
2. Instalar dependencias: `npm install`

## Ejecutar los tests

```bash
npm run test nombre-del-archivo.test.js
```
